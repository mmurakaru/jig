# Implement the fix

Change the code so the red test goes green. Fix the root cause, keep the
change minimal, and never weaken or delete a test. A previous attempt's
handoff tells you why it failed - revise the approach, do not repeat it.

Done when you have run the test and seen it green for the right reason:
your summary names the root cause and why this change removes it; changed
files as artifacts. escalate = the fix needs a decision only a human can
make.
